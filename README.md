# thealxlabs.ca

Toronto-based software lab building AI tools, developer utilities, and open-source experiments.

## Tech Stack
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/anomalyco/thealxlabs.ca.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
